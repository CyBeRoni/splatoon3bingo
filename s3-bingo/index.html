Splatoon 3 Weapons Bingo

Marco's Stream Version

This Weapons Bingo Card generator is based heavily on that by Go1den, but modified for visuals.

To use, create an OBS Browser source (minimum size: 1920x1080) and enter the URL. If custom positioning is desired, do this in a browser and copy the resulting URL after, as the Browser Source will not save this between launches of OBS. Crop out the settings area in OBS. Right-click and choose "interact" to be able to press the buttons.

Tips:

  • Making the browser source 2300 pixels wide allows you to position the settings area off-screen
  • If you keep the "seed" parameter in the URL, the board will start with the same contents every time. Removing it will let OBS generate a new board every time you launch it.
  • If you hide the weapon randomizer and click either of the randomizers's buttons, it will show for 10 seconds and then hide again.
  • To avoid having to use "interact", there's an OBS dock you can add as well.
